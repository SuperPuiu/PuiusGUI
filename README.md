# PuiusGUI
This project aims to give an easy to use, single file GUI library to use along with SDL2.

Contains:
- GUI Buttons;
- GUI Labels;
- GUI ImageLabels;
- Input handling function;

# Requirements
To build, PuiusGUI.c requires SDL.h and SDL_image.h and SDL_ttf.h.
